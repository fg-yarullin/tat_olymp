# Загрузка сканов работ муниципального этапа

Сканы работ нужны для онлайн-показа: после публикации результатов участник заходит на показ
по ФИО и дате рождения и видит свою проверенную работу. Каждый скан привязывается к работе
**по шифру участника**.

Загрузить сканы можно двумя способами:

- **через сайт** — кнопка «Сканы (ZIP)» (до **256 МБ**) — для обычных объёмов;
- **через сервер** — для **больших архивов (несколько ГБ)**, когда загрузка через сайт не проходит.

Ниже — про загрузку через сервер.

---

## Шаг 0. Подготовьте файлы

1. Каждой работе должен быть присвоен **шифр** (это делает муниципальный координатор на странице
   «Состав»).
2. Назовите файл скана **шифром участника**, например:

   ```
   A-001.pdf
   A-002.pdf
   A-014.jpg
   B-103.png
   ```

   - один файл = одна работа (многостраничную работу соберите в один PDF);
   - форматы: **pdf, jpg, png**;
   - папки внутри архива не важны — учитывается только имя файла.

3. Сложите все файлы в один ZIP-архив **или** в одну папку.

---

## Шаг 1. Узнайте ID олимпиады

Откройте в админке раздел **«Олимпиады»** → вкладка **«Муниципальный»**. Рядом с названием
предмета указан номер вида **#12** — это и есть ID олимпиады.

---

## Шаг 2. Положите файл на сервер

Скопируйте архив или папку со сканами на сервер (по SFTP/SCP) в любую папку, например:

```
/home/u/uploads/scans_fizika.zip
```

или распакованную папку:

```
/home/u/uploads/scans_fizika/
```

> Большой архив надёжнее **распаковать в папку** прямо на сервере и указать путь к папке —
> файлы читаются по одному, без распаковки гигабайтов в память.

---

## Шаг 3. Запустите команду

В каталоге проекта выполните (замените `12` на ID вашей олимпиады и укажите свой путь):

```bash
# из ZIP-архива
php artisan scans:import 12 /home/u/uploads/scans_fizika.zip

# или из папки
php artisan scans:import 12 /home/u/uploads/scans_fizika/
```

В ответ команда покажет, сколько сканов загружено и какие файлы пропущены, например:

```
Олимпиада: Физика (#12). Работ с шифром: 230.
Загружено сканов: 228.
Пропущено: 2
  • C-500.pdf: шифр «C-500» не найден среди работ
  • A-014.txt: недопустимый формат (нужен pdf/jpg/png)
```

Повторный запуск можно делать сколько угодно — ранее загруженные сканы не теряются, новые
добавляются/обновляются.

---

## Возможные сообщения

| Сообщение | Что значит |
|---|---|
| `Олимпиада не найдена` | Неверный ID — проверьте номер `#…` в админке |
| `Загрузка сканов доступна только для муниципального этапа` | Указана школьная/региональная олимпиада |
| `Путь не найден` | Неверный путь к файлу/папке на сервере |
| `У участников этой олимпиады не заданы шифры` | Сначала присвойте шифры на странице «Состав» |
| `шифр «…» не найден среди работ` | Имя файла не совпало ни с одним шифром этой олимпиады |

После загрузки опубликуйте результаты (кнопка «Опубликовать»), чтобы открыть онлайн-показ.
