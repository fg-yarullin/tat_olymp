# FAQ — для председателя предметной комиссии

**Что я вижу в кабинете?**
Раздел «Проверка работ МЭ» — назначенные вам олимпиады. Работы показаны **обезличенно**: шифр,
класс, класс участия, первичный балл. ФИО, школа и АТЕ не отображаются.

**Какие работы мне доступны?**
Только зашифрованные работы вашего АТЕ по тем олимпиадам, на которые вас назначил муниципальный
координатор.

**Как ввести первичный балл?**
В строке работы — ввод балла вручную. Если у олимпиады задано количество заданий, баллы вводятся
по заданиям, а первичный балл считается как их сумма.

**Можно ли ввести баллы массово?**
Да. Скачайте «Шаблон баллов (XLSX)», заполните колонку «Балл» и загрузите файл. Баллы сопоставляются по шифру;
строки с неизвестным шифром, дублем или баллом выше максимума пропускаются — список покажется
после загрузки.

**Почему ввод закрыт?**
Окно ввода первичных результатов закрывается по сроку. Срок виден на странице («ввод открыт
до …»). Изменение срока — у администратора.

**Где найти шифр участника?**
Поиск по шифру и фильтры по классу/классу участия — в шапке списка работ.
