# FAQ — для школьного оператора

**Куда попадаю после входа?**
На «Главную» — обзор ближайших олимпиад и сроков. Разделы кабинета — в верхнем меню:
«Моя школа», «Учащиеся», «Результаты», «Приглашения на МЭ».

**Как добавить учащихся?**
Раздел «Учащиеся» — вручную по одному или массово импортом (по шаблону). Выбывших учеников
помечайте «выбыл», вернувшихся — «восстановить».

**Как внести результаты школьного этапа?**
Раздел «Результаты» → выбрать олимпиаду → ввод балла каждому участнику вручную или массовым
импортом. Если задано количество заданий — баллы вводятся по заданиям, итог считается сам.

**Почему ввод результатов закрыт?**
Ввод закрывается автоматически по сроку, заданному администратором. Срок и его продление видны
на странице результатов («ввод открыт до …»). Продлевает срок администратор.

**Как расставить призёров?**
Если администратор задал пороги, появляется расстановка статусов по баллу: победитель ставится
вручную, призёр — по порогу. Победителя система при пересчёте не сбрасывает.

**Что с приглашениями на муниципальный этап?**
Раздел «Приглашения на МЭ» показывает учеников вашей школы, прошедших на МЭ. Списки формирует
муниципальный координатор по итогам ШЭ.

**Показ работ участникам.**
На школьном этапе отдельного онлайн-показа нет — работы участники смотрят в рамках своей школы.

Не нашли ответ? Обратитесь к муниципальному координатору или администратору.
