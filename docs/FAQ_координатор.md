# FAQ — для муниципального координатора

**С чем я работаю?**
С муниципальным этапом по своему АТЕ: формирование состава, шифры, результаты, председатели
комиссий, протокол. Разделы — в меню «Муниципальный этап» и «Председатели комиссий».

**Как сформировать состав участников МЭ?**
Страница «Состав» → «Сформировать из ШЭ / прошлого МЭ»: добавляются призёры/победители ШЭ с баллом
не ниже порога (порог задаёте вы по классам) и прошлогодние призёры МЭ. Можно добавить вручную
«по ходатайству» или участника «из другого региона».

**Что такое шифр и как его присвоить?**
Шифр — обезличенный код работы для проверки председателем и онлайн-показа. На странице «Состав»:
вручную в строке участника или массово — скачать шаблон (XLSX), проставить шифры в колонке «Шифр» и загрузить этот же файл.

**Как загрузить сканы работ?**
Кнопкой «Загрузить сканы работ (ZIP)» на странице «Состав» (по своему АТЕ). Имена файлов = шифры.
Подробно — в инструкции [«Загрузка сканов работ»](/help/scans), включая способ для больших архивов.

**Как назначить председателя комиссии?**
Раздел «Председатели комиссий» → в строке олимпиады «+ Председатель»: создаётся пользователь и
сразу привязывается к этой олимпиаде по вашему АТЕ.

**Кто вводит баллы МЭ?**
Председатель комиссии (обезличенно по шифру) и/или вы — на странице «Результаты». Итог считается
автоматически (первичный балл + добавка по апелляции).

**Можно ли продлить срок ввода?**
Продление делает администратор. Эффективные сроки (с учётом продлений) видны в «Расписании».

**Как выгрузить протокол МЭ?**
Кнопка «Протокол МЭ (XLSX)» на странице «Результаты» (если администратор настроил шаблон протокола).
