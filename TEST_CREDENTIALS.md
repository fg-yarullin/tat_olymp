### Привязки (выбираются динамически из засеянного каталога)

- **Супер-координатор** — привязан к казанской районной АТЕ (АТЕ с наибольшим числом МСУ).
- **Муниципальный координатор** — первая обычная (не казанская) АТЕ, по умолчанию Агрызский район.
- **Школьный оператор** — школа внутри АТЕ муниципального координатора.

> Примечание: в исходном каталоге Казань разбита на **районные АТЕ** (городской округ = несколько АТЕ, у каждой по нескольку МСУ); единой «Казань-АТЕ» нет.

## Гостевой доступ (без логина)

**Онлайн-показ работ:** `/showcase` (или ссылка «Онлайн-показ работ» на главной).
Участник входит по **ФИО + дате рождения** — отдельной учётной записи нет.
Для проверки нужен ученик с опубликованными результатами (создаётся в рамках сценариев школьного/муниципального этапа).

## Как пересоздать аккаунты

```bash
# Полный пересев (сбрасывает БД):
php artisan migrate:fresh --seed --force

# Только демо-пользователи (без сброса остальных данных):
php artisan db:seed --class=DemoUsersSeeder --force
```
